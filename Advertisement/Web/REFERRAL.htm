我先說給你聽為什麼會有這種的賺錢方式
在以前INTERNET剛開始發熱時,美國有許多的公司便看上了電子行銷這塊大餅
依據統計指出...寄廣告件是以大約1000:2的成功率來產生賺錢機會的
也就是說...當你用電子信箱來宣傳你的商品...每一千個人就會有兩個人會買
每一百萬個人會有2000個人買..如果你只靠收集email和寄email這種方式
就多讓2000多人買你的東西..那你做不做...
何況如果用寄DM的話...光郵資就貴到嚇死人了
如果一封信郵資只要一元...寄給一百萬人就要一百萬元...
而用e-mail來做,只要有安裝網路....只要幾分鐘就搞定了

所以在那時候就有很多公司開始收集e-mail然後寄廣告件
但是由於造成了許許多多的人每天幾乎都收到100多封的廣告信
因此美國政府制定了法律規定...若遭檢舉寄廣告信, 一封廣告信必須付出250美元罰金
所以接下來就有一些網路行銷公司的成立...受其它公司委託發廣告
由我們這些平民來加入..自願收到廣告信
或是在螢幕上開啟一個視窗來顯視廣告
而在我們看廣告時,就有一定比例的酬勞可以拿......所以申請加入是免費的

cashsurfers這家公司從1999年成立至今...信用可以說是目前最好的..

而且也停止了對亞洲及澳洲的支票支付....
所以在一些賺錢討論的地方...紛紛指出了cashsurfers這家新公司的好處

另外你必須拉下線來替你賺錢...不然靠自己是賺不了多少的(一個月最多25美元)

下線的意思就是經由你的介紹而加入的人....
你可以從他們的點數中抽取一定比例的點數(但他們點數不會減少)
這樣下線如果多...點數的累積就很快 點數多...錢就多
現在平均大約是使用viewbar每個小時可以拿到$0.1至$0.2美金
就是你一個月用120小時...可以用$12至$24美金可以拿
這只是你靠你自己賺的 有下線的話就更多

舉個例子...如果你有五個很用功的下線,而他們也都有五個很用功的下線
他們下線的下線也有五個...下線的下線的下線也有五個...那麼
你 $12
第一層下線:5 x $12 x 25% = $15
第二層下線:25 x $12 x 10%- = $30
第三層下線:125 x $12 x 10% = $150
第四層下線:625 x $12 x 10% = $750
總計 : $957 / 月
(最重要的就是他們要用功才行)

你一個月大約有957元美金可以拿 大約31581台幣
但要在短期內造成這種下線系統是不可能的
而且也不是每個你的下線都那麼用功...所以不要以為一定可以那麼賺錢
在一段時間後...一個月有個8000元台幣左右就很不錯了
